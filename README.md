## :wrench: Configuration

`Node version: v16.19.0`

```bash
1. Create .env file 
$ cp .env.example .env

2. Add the api url
e.g: http://localhost:3000

3. Install all dependencies
$ npm install

4. Run the project
$ npm start
```
